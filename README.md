# Diga sim a adoção — Site Institucional

Projeto completo para entrega acadêmica com foco em adoção animal.
Inclui 3 páginas semânticas, design system, responsividade, componentes UI, JS modular com SPA básico,
validação de formulários, acessibilidade (WCAG 2.1 AA) e instruções de Git/GitFlow e deploy.

## Estrutura
- index.html, projetos.html, cadastro.html
- css/, js/, assets/, docs/

## GitFlow sugerido
Branches principais: main (produção), develop (integração).
Use feature/*, release/* e hotfix/* conforme GitFlow.
Commits semânticos: feat:, fix:, chore:, docs:, style:.

## Deploy sugerido
- GitHub Pages: publicar branch main
- Netlify/Vercel: conectar repositório
